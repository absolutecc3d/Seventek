# Seventek Website

A responsive static landing page for a Discord tools and server development company.

## Files
- `index.html` - Page structure and content
- `styles.css` - Responsive design and animations
- `script.js` - Navigation, scroll effects, and demo form behaviour

## Run locally
Open `index.html` in a browser.

For best results, use a local development server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Before publishing
- Replace the placeholder Discord and GitHub links in the footer.
- Connect the contact form to a backend, Formspree, Netlify Forms, or another form service.
- Replace example project names and content with real Seventek products if needed.
